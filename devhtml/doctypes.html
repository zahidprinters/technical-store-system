Advanced Features & Capabilities

Auto-Discovery Pattern

Revolutionary installer that automatically discovers and installs DocTypes. No manual updates needed - just add files to setup/doctypes/ and run migrate.

6-Level Location Hierarchy

Intelligent warehouse structure: Store → Zone → Area → Rack → Shelf → Bin. Auto-generates codes like WH-1-Z-A-R01-S01-B01 with cascading dropdowns.

Tree Structure Support

Unlimited nesting for item groups with inheritance. Real-time statistics and configuration inheritance from parent to child groups.

Advanced Tracking

Flexible serial and batch number tracking. Individual item control, warranty management, expiry dates, and quality control.

Multi-UOM Support

27+ predefined units with automatic conversions. Primary and alternative UOMs with validation and intelligent calculations.

Enterprise Security

Comprehensive role-based access control, audit trails, and data protection. Field-level permissions and secure API endpoints.

Implementation Example

# Auto-Discovery Installer Pattern # File: setup/doctypes/YourDocType.py doctype = { "name": "Your DocType", "fields": [ { "fieldname": "field_name", "fieldtype": "Data", "label": "Field Label", "reqd": 1 } # Add more fields... ], "permissions": [ { "role": "Store Manager", "read": 1, "write": 1, "create": 1 } ] } # That's it! The system automatically discovers and installs this DocType # when you run: bench --site your-site.local migrate
